[LeetCode]199 二叉树的右视图

2019-02-15

题目描述

给定一棵二叉树，想象自己站在它的右侧，按照从顶部到底部的顺序，返回从右侧所能看到的节点值。

示例

输入: [1,2,3,null,5,null,4]
输出: [1, 3, 4]
解释:
       1            <---
     /   \
    2     3         <---
     \     \
      5     4       <---

思路

  • 层序遍历
  • 先遍历右子树, 后遍历左子树
  • 第一次遇到下一层的就记录下来
